# Agents

## Project Context

This project generates an OpenSCAD model of a large cube assembled from 4 smaller cubes (1cm per side) in a pink/purple checkerboard pattern.

## Key Facts

- Units: millimeters (10mm = 1cm)
- Small cube size: 10mm × 10mm × 10mm
- Layout: 2×2×2 grid (8 cubes total, true cube shape)
- Colors: DeepPink and Purple, alternating checkerboard
- Main file: `cube.scad`
